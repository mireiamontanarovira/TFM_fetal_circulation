# Fetal circulation mechanistic model

Mechanistic model of fetal circulation and cardiovascular development across gestational age.

This project is based on the 0D fetal circulation model presented in: Villanueva-Baxarias et al., 2025, which provides a reference parametrization for a 36-week gestational age fetus.

The present framework extends the original model by implementing gestational-age-dependent scaling laws to simulate cardiovascular development between 24 and 40 weeks of gestation.

The code includes:
- Gestational-age-dependent parameter scaling
- Fetal weight and percentile-based adaptation
- Placental growth scaling
- Hemodynamic simulations of the fetal circulation
- Cardiac pressure, volume, and flow analysis
- Doppler-derived indices (MPI, E/A, PI, ETF, FTF)
- Vessel velocity and wall shear stress computation
- Validation against fetal echocardiography and Doppler literature

The framework is intended to study normal fetal cardiovascular physiology and developmental hemodynamics across gestation.
